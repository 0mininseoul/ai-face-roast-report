## 0. 플레이북 한 페이지 요약

> **얼평으로 트래픽을 펌프업하고, 같은 얼굴 분석 엔진 위에 "진지한 분석" 카테고리를 얹어서 결제를 받는다.**

- 풍자 콘텐츠는 본질적으로 결제 전환율이 낮지만 **공유율은 압도적으로 높다**.
- 진지 분석(첫인상/관상/이미지 컨설팅)은 공유율은 낮지만 **결제 의향이 5~10배 높다**.
- 둘은 **같은 엔진**(MediaPipe 478 랜드마크 + Gemini Vision + 메트릭 산출 + 톤 전환)을 공유한다.
- 따라서: **얼평 = 무료 트래픽 펌프**, **진지 분석 = 유료 매출 펌프**, 둘이 한 사이트 안에서 cross-sell.

핵심 risk 3가지: (1) 콘텐츠 신고로 도메인/계정 정지, (2) 트래픽이 못 터져 펌프 자체가 안 도는 것, (3) 진지 모드의 정확도 신뢰성 문제. 각각의 대응은 §7, §10에 명시.

## 1. 포지셔닝

### 1.1 한 문장 정의
> **"한국어 인터넷 밈 톤으로 무자비하게 외모를 풍자하는, 실시간 AI 얼굴 분석 쇼."**

이건 "얼굴 평가 앱"이 아니라 **AI가 주연인 한국어 풍자 엔터테인먼트**다. 사용자는 결과를 진지하게 받아들이지 않고, **친구들에게 보여주기 위한 짤거리**로 소비한다. 이 인식이 흐려지면 (a) 진짜 상처받은 사용자 / (b) 광고주 거부감 / (c) 플랫폼 신고가 동시에 터진다.

### 1.2 인식 슬로건 후보
- "AI도 욕하다 지치는 면상 분석기"
- "AI가 너 보고 한숨 쉬는 서비스"
- "AI 얼평보고서 — 면상은 못 숨긴다"

### 1.3 무엇이 *아닌가* (전략적 거리두기)
- 외모 평가 점수 사이트 ❌ (페이스 점수 매기는 사이트들과 명확히 분리)
- AI 미용 컨설팅 ❌ (성형/뷰티 광고 붙이는 순간 톤 깨짐)
- 진지한 관상 사이트 ❌ (얼평 본진 안에서는)

→ 진지 카테고리는 **별도 제품/별도 브랜드 또는 별도 모드**로 분리해야 한다. (§9 참조)

## 2. 핵심 타겟

### 2.1 1차 타겟 — 20대 남성 (현재 톤 그대로 fit)
- **연령**: 만 18~28세
- **채널**: 디시인사이드(특히 외모지상주의·아햏햏 갤), 에타, 페이스북 짤방 페이지, 트위터(X), 단톡방
- **JTBD (Jobs-To-Be-Done)**:
  1. 단톡방에 던질 짤거리가 필요하다
  2. 자학 유머로 친구들 웃기고 싶다
  3. AI가 욕하는 게 신기하고 웃기니 봐보고 싶다
  4. 친구가 보낸 링크를 안 따라가면 단톡방 흐름에서 빠진다 (FOMO)
- **수용 톤**: 현재 PRD §7.1 voice B 톤 그대로 OK. 디시/일베 밈 어투 자연스러움.
- **결제 의향**: **낮음** (~1~3% 전환). 풍자 자체는 유료화하면 안 됨.

### 2.2 2차 타겟 — 20대 여성 (톤 조정 필요)
- **연령**: 만 18~28세
- **채널**: 에타, 인스타 스토리, 트위터, 더쿠, 카톡 단톡방
- **JTBD**:
  1. 친구랑 같이 깔깔거리며 돌려보기
  2. 인스타 스토리 짤로 쓰기 ("AI가 나한테 한 말 ㅠㅠㅠ")
  3. **자기관찰 욕구** (어떻게 보일까)
- **수용 톤**: 현재 톤은 위험. **셀카/필터/SNS 디스 중심으로 재조정**, 외모 직격 강도는 낮추되 디테일 풍자(MBTI 추정, 관상 코멘트, "친구들이 사진 안 보내는 이유" 류)를 강화. 인종/체형/특정 외모 요소 직격은 빼야 신고 안 깔림.
- **결제 의향**: **중간** (3~7%) — 특히 진지 모드(첫인상, 매력도 진지 분석)에서 의향 높음.

### 2.3 3차 타겟 — 글로벌 영어권 (Phase 3 prototype)
- 후순위. Phase 3에서 영문 prototype만 띄움. 단, **한국어 어투의 마법은 번역 안 됨** — 영문판은 처음부터 다른 톤으로 재설계해야 한다 ("English roast culture": 미국 stand-up 디스 톤). 단순 번역 금지.

### 2.4 타겟별 톤 분리 (실행 단위)
| Segment | 진입 페이지 분기 | mainCopy 톤 | §5 본문 강도 | 광고 노출 |
|---|---|---|---|---|
| 20대 남성 (현재) | 기본 | 직격 욕설 OK | 풀 강도 | 가능 |
| 20대 여성 | 성별=여 + 향후 강도 토글 | 셀카/필터/관계 디스 위주 | 한 단계 약 | 가능 |
| 진지 모드 (B 트랙) | 별도 진입 | 욕설 0 | 컨설팅 톤 | 광고주 friendly |

## 3. 핵심 니즈 (JTBD)

### 3.1 1차 니즈 (모든 사용자 공통)
> **"단톡방에 던졌을 때 사람들이 반응할 콘텐츠가 필요하다."**

이 니즈가 다른 모든 니즈를 포괄한다. 즉 모든 제품 결정은 다음으로 환원된다:
- 결과 페이지를 받은 사람이 카톡에 던지고 싶어지는가?
- 받은 사람이 클릭해서 들어오는가?
- 들어와서 자기 것을 만들고 싶어지는가?

### 3.2 2차 니즈 (segment별)
- **남성**: 자학 + 친구 디스 ammo
- **여성**: 자기관찰 + 인스타 스토리 짤
- **진지 모드 사용자(B 트랙)**: 자기 매력 점검, 면접 인상 점검

### 3.3 절대 충족하지 말아야 할 가짜 니즈 (트랩)
- "객관적인 외모 점수" → 메디컬/외모지상주의 사이트와 같은 카테고리에 빠짐. moat 잃음.
- "성형 추천" → 광고주 받기 시작하면 톤 무너짐.
- "유명인과의 닮은꼴" → 초상권 리스크.

## 4. 차별화 / Moat

### 4.1 사용자가 인식하는 차별점 (본인 답변 기반)
1. **한국어 고수위 풍자 톤** — 한국어 인터넷 밈/욕설 어투를 자연스럽게 구사
2. **실시간 분석 UX** — 단순 이미지 업로드형이 아니라 카메라 위에 메쉬+카드가 얹히는 "AI 시스템" 인상

### 4.2 Defensibility 솔직 평가
| 차별점 | 진짜 moat인가? | 모방 난이도 | 유효 기간 |
|---|---|---|---|
| 한국어 고수위 톤 | **약한 moat** — 프롬프트만 베끼면 1주 내 복제 가능. 단 톤 *감각*은 베껴도 어색해질 가능성 큼 | 낮음 | 3~6개월 |
| 실시간 메쉬 UX | **중간 moat** — 구현 비용 + 디테일 디자인 톤 합성이 의외로 어려움 | 중간 | 6~12개월 |
| 도메인/SEO/검색량 선점 | **시간 moat** — 첫 6개월 내 "AI 얼평" 검색어 장악 가능 | 시간 들임 | 12개월+ |
| 카톡 공유 회로 + 시드 사용자 망 | **약한 moat** | 낮음 | 일시적 |

→ **결론**: 본 제품의 moat는 **"속도와 시간"**이다. 한국어 풍자 카테고리에서 "AI 얼평" = 우리 = 검색 1위 = 추천 1위라는 인식을 6개월 안에 선점하는 것이 유일한 진짜 해자. 이 시간 안에 카테고리 확장(B 트랙)도 끝내야 함.

### 4.3 본 플레이북의 mantra
> **"6개월 안에 카테고리 인식을 굳히고, 같은 시간에 B 트랙으로 매출 다리를 놓는다."**

## 5. Phase별 성장 전략

### Phase 0 — 시드 화제성 측정 (D+0 ~ D+14, **지금**)
**목표**: 현 제품이 바이럴 신호를 내는지 객관적으로 확인.

**행동**:
1. **채널 다양화 시드 배포** (현재 카톡/쓰레드/에타/인스타 외 추가):
   - 디시 외모지상주의 갤, 아햏햏 갤 — 짤 형태로 "AI가 나 보고 한 말 ㅋㅋㅋ" 1인칭 글
   - 더쿠 (여성 사용자 톤 측정 목적)
   - 페북 페이지 짤 계정 (운영자 콜드 메시지로 1~2곳)
   - 트위터(X) — 본인 계정에서 결과 캡쳐 + 짧은 코멘트
2. **단 1개 측정 지표만 본다**: **세션당 카톡 공유 클릭률 (K-share)**
   - 카톡 공유 버튼 클릭 / 결과 페이지 도달 세션 수
   - 목표: 15%+ (이 수준이면 바이럴 루프가 자연 회전 가능)
3. **금기**: 광고비 1원도 쓰지 않음. 유료 인플루언서 NO. 시드 단계는 **organic K-factor 측정 단계**다.

**Go/No-Go 결정 (D+14)**:
- K-share ≥ 15% + 일 신규 세션 100+ → Phase 1로 진입
- K-share < 10% → 바이럴 루프 자체를 재설계 (§5 mainCopy 톤, 공유 카드 디자인)
- K-share 10~15% → Phase 1 진입은 하되 §5.1.2 작업 우선순위 1번으로

### Phase 1 — 바이럴 루프 hardening (D+15 ~ D+60, 약 1.5개월)
**목표**: 일평균 1,000 세션, K-share 20%+ 안정화.

**핵심 작업 (우선순위 순)**:
1. **결과 페이지 동적 OG 이미지** — 카톡 공유 미리보기에 그 사람의 mainCopy + 얼굴 썸네일이 박혀 나오게. 현재 PRD는 정적 OG. *카톡 미리보기에 얼굴 + 욕설 한 줄이 보이는 것*이 클릭률을 결정한다. (Vercel OG image API + Supabase signed URL 합성)
2. **mainCopy 캡쳐 공유 최적화** — 결과 페이지 상단을 인스타 스토리 9:16 또는 정사각 1:1로 한 번에 캡쳐 가능한 "공유용 카드" 모드 추가. 우상단 [짤 저장] 버튼.
3. **"친구 분석시켜봐" 강 CTA** — 결과 페이지 하단에 "이 친구도 한 번 까봐줘" 카톡 링크 공유 버튼. 메시지 prefilled.
4. **재방문/세션 재생성 마찰 제거** — [다시 분석] 버튼 클릭 시 카메라 권한 재요청 없이 즉시 재진입.
5. **모바일 fallback → 모바일 결과뷰어 풀지원** — 현재 PRD는 모바일에서 분석 불가. 분석은 데스크톱 전용 유지하되, **결과 페이지(`/result/[id]`)는 모바일에서 완전히 보여야 한다**. 카톡 링크 받는 사람의 90%가 모바일이기 때문.
6. **SEO 기반 토대** — `/`, `/result/[id]`(만료 후 페이지) 메타 태그, sitemap, naver/google search console 등록. "AI 얼평" 검색어 점유 시작.

**금기**: 이 단계에서 결제 시스템 만들면 안 된다. 트래픽 신호가 안 잡힌 상태에서 결제는 시간 낭비.

**Go/No-Go (D+60)**:
- 일 1,000 세션 + K-share 20%+ → Phase 2 진입 (수익화 시작)
- 일 200~1,000 세션 → Phase 1 연장 (채널 확장 추가)
- 일 200 미만 → **PMF 결여**. 콘셉트 재정의 또는 셧다운 검토

### Phase 2 — 수익화 실험 (D+61 ~ D+120, 약 2개월)
**목표**: 첫 매출 발생, 6개월차 월 100만원 ARR 경로 검증.

**3트랙 병렬 (단, 시간 배분 우선순위)**:

**트랙 A (50% 시간) — 결제 unlock 1개 출시**
- **MVP 후보 1**: "심층 분석 PDF + 영구 보관" — ₩1,900
  - 24h 만료 풀어주고 PDF 다운로드 (브라우저 print → PDF면 됨)
  - 마찰 가장 낮음, 결제 의향 검증용
- **MVP 후보 2**: "친구 비교 분석" — ₩2,900
  - 두 명 결과를 한 페이지에 비교 + AI가 더 못생긴 쪽 판정
  - 카톡 단톡방 화제성 폭발력 큼
- **결제 수단**: 토스페이먼츠 또는 카카오페이 결제. 멤버십/구독 NO, 1회 결제만.
- **선출시 1개 후보 선정 기준**: 후보 2가 바이럴 추가 동력까지 만든다. 시간 충분하면 후보 2 우선.

**트랙 B (40% 시간) — 진지 모드 prototype 1개 출시 (B 트랙의 첫 다리)**
- 후보: **"AI 첫인상 분석 (취준 / 소개팅 대비)"**
  - 같은 엔진 (MediaPipe + Gemini Vision)
  - 톤만 진지 컨설팅으로 교체
  - 별도 URL: `/first-impression` 또는 별도 도메인
  - 결제: ₩4,900 (결제 후 분석)
  - 결제 의향이 얼평 unlock보다 **5~10배 높음** — 실용 가치가 명확
- 출시는 작게: 진입 페이지 + 분석 + 결과 페이지 1개씩만. 라이브 카드 UX는 재활용.

**트랙 C (10% 시간) — 광고 수익 (passive)**
- 결과 페이지 하단(스크롤해서 §3 아래쯤)에 Google AdSense 1슬롯
- 카카오 애드핏 1슬롯도 가능
- 단, 톤 fit 안 맞는 광고(성형, 다이어트, 미용) 카테고리는 차단 설정
- 의의: 시간을 거의 안 쓰고 트래픽 → 매출 1차 변환

**Phase 2 종료 시점 매출 가정**:
- 일 2,000 세션 × 결제 전환 1.5% × ₩2,500 = ₩75,000/일 = **월 ₩2,250,000**
- 광고 ₩200,000 ~ 500,000/월
- B 트랙(첫인상) 일 30 결제 × ₩4,900 = ₩147,000/일 = **월 ₩4,400,000**
- 합산 가정 시 월 300~700만원 가능. 단, 위는 낙관 가정. 실제로는 50~30% 수준이 정상.

### Phase 3 — 수익 모델 lock-in + 글로벌 prototype (D+121 ~ D+180, 약 2개월)
**목표**: 가장 잘 도는 1~2개 매출 라인에 집중, 나머지는 정리. 글로벌 prototype 띄우기.

**행동**:

## 9. B 트랙 — 인접 카테고리 확장 로드맵

본 트랙이 **장기 매출 다리**다. 풍자(얼평)는 6개월 후 트래픽이 식기 시작할 가능성이 크다(밈 수명). 그 전에 같은 엔진 위에 진지 라인을 깔아두는 것이 핵심.

### 9.1 엔진 재사용 가능 자산
- MediaPipe 478 랜드마크 + 메트릭 산출 코드
- Gemini Vision + 시스템 프롬프트 골격 (톤만 swap)
- 라이브 분석 UX 카드 컴포넌트
- 결제·결과 페이지 인프라 (Phase 2에서 깔리면 그대로 재사용)

### 9.2 카테고리 후보 (출시 우선순위)
1. **AI 첫인상 분석** (Phase 2 트랙 B) — 취준/소개팅 대비. ₩4,900. **결제 의향 1순위**.
2. **AI 매력도 자가진단** — 진지 톤. 여성 타겟 강함. ₩3,900.
3. **AI 관상 분석** — 사주식 톤. 30대~40대 흡수 가능. ₩4,900.
4. **AI 커플 궁합** (얼굴 매칭) — 두 명 분석 + 매칭 점수 + 코멘트. ₩5,900. 바이럴 동반.
5. **AI 면접 인상 분석** — B2C 취준 시장. ₩6,900. 단가 가장 높음.

각 카테고리는 별도 진입 페이지 1개씩, 같은 엔진 호출, 톤만 다른 시스템 프롬프트, 같은 결제 흐름. **신규 카테고리 출시 비용 = 시간 1주~2주**.

### 9.3 브랜드 분리 여부
- **권장: 같은 도메인 + 다른 모드**. 별도 브랜드 만들면 시간 소비 큼.
- 메인 페이지에서 "얼평" 또는 "진지 분석" 모드 선택 토글. 진지 모드 진입 시 욕설/메시 톤 모두 컨설팅 톤으로 swap.

  - 신고 임계치(§10.3) 초과 시 톤 다운 토글 즉시 가동
  - 사용자가 결과 페이지 자신의 것 즉시 삭제 요청할 수 있는 셀프 UI를 Phase 1에 추가 (현재 PRD는 이메일 요청만 — 갈등 키울 위험)

### 10.2 플랫폼 리스크
- **카카오 SDK 키**: 톤 문제로 카카오 비즈 채널 정지될 수 있음. 백업 공유 채널(URL 복사) 항상 유지.
- **Vercel/Supabase**: 신고 누적 시 계정 정지 가능성. 정기적으로 약관 위반 트리거 단어 모니터링 (§10.3).

### 10.3 모니터링 임계치 (자동 알람)
- 일 신고 메일 ≥ 5건 → 톤 한 단계 다운 검토
- 사용자 "삭제 요청" ≥ 일 10건 → §10.1 셀프 삭제 UI 즉시 출시
- 언론 부정 보도 1건 이상 → 24시간 내 공식 입장문 + 일시적 톤 다운 모드

### 10.4 비용 폭주 리스크
- Gemini 비용 세션당 35원 가정. 일 5,000 세션이면 일 ₩175,000. 매출이 안 따라오는데 트래픽만 터지는 시나리오 존재.
- **대응**: IP rate limit (이미 PRD §11.4) + 일일 Gemini API 비용 cap 환경변수 추가, 초과 시 라이트 모드(Flash only)로 자동 fallback.

### 10.5 의사결정 보류 (운영자가 향후 결정 필요)
- 사업자등록 시점 (Phase 2 결제 출시 직전 필수)
- 도메인 — 현재 도메인 정해졌는지 미확인. 짧고 기억하기 쉬운 .com 또는 .kr 권장
- 회사 명의 SNS 운영 여부 — 운영자 본인 노출 여부 결정 필요
- 진지 모드(B 트랙) 별도 브랜드 분리 여부 — Phase 2 트랙 B 출시 직전 재결정
